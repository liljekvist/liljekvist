C++ Website

This project is not done and is still in development

Im working on a concept website together with a friend. It has a backend written in C++ and a frontend using html, css and javascript. We have divided the work into frontend and backend. Im doing the backend part. The vision is that it's a forum that is focused on privacy. There is no register- or login-system, instead its using cookies to identify a user. We dont save any information about the user other than a random string we call Unique User Identifier (UUID) and a random int User Identifier (UID). The cookie we send to the user is the UUID and is the translated into the UID, the UID is then viewed almost as a username on the website. To write the backend I am using a framework called Drogon.
